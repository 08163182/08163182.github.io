生成器

[]–>()可创建generator(可迭代对象)
斐波拉契数列：

def fib(max):
    n, a, b = 0, 0, 1
    while n < max:
        yield b
        a, b = b, a + b
        n = n + 1
    return 'done'

genator调用next()执行,遇yield语句中断返回迭代值。再次执行从返回的yield下一条继续执行。得到返回值捕获StopIteration错误
range(n)产生（0-n）的数，不包括n，range(0)无输出 。

list/dict/str是Iterable不是Iterator,通过iter()可获得Iterator对象
for循环本质通过调用next()函数实现

函数式编程

高阶函数

变量可以指向函数；
abs() abs是函数本身，abs()是函数调用。
函数名也是变量；
abs是变量指向一个计算绝对值函数
传入函数;
函数作为参数传入,这样的函数称为高阶函数。

map/reduce

map(函数，Iterable序列)
函数作用于序列每个元素，结果作为新的Iterator返回。
reduce(f,[x1,x2,x3,x4]) = f(f(f(x1,x2),x3),x4)
reduce函数接收两个参数并把结果和序列的下一个元素做累积计算。

filter

filter(函数，Iterable序列)
函数作用于序列每个元素，根据结果为True或者False筛选。
strip()移除字符串头尾指定字符。
求素数的函数：

def primes():
    yield 2
    it = _odd_iter() # 初始序列
    while True:
        n = next(it) # 返回序列的第一个数
        yield n
        it = filter(_not_divisible(n), it) # 构造新序列

sorted

sorted高阶函数，可对list进行排序。
sorted(['bob','about','Zoo'],key=str.lower,reverse=True)

返回函数

函数作为返回值：

def lazy_sum(*args):
    def sum():
        ax = 0
        for n in args:
            ax = ax + n
        return ax
    return sum

f是求和函数；f()返回最终结果；返回一个函数时函数并未执行；每次调用都返回新的函数；
闭包：返回函数不要引用任何循环变量或后续会发生变化的变量。
内部函数可以修改外部list中的元素，内部函数修改外部函数参数需声明nonlocal.

装饰器

decorator:

@log('execute')
def now():
    print('2016')
import functools

def log(text):
    def decorator(func):
        @functools.wraps(func)
        def wrapper(*args, **kw):
            print('%s %s():' % (text, func.__name__))
            return func(*args, **kw)
            return wrapper
    return decorator

加上@functools.wraps()可把原始函数的name属性复制到wrapper函数中。

偏函数

参数太多，functools.partial可创建新的函数，可固定原函数的部分参数。偏函数可接受函数对象，*args,**kw 3个参数。

@propetry

@propetry装饰器可把方法变成属性调用
定义只读属性 脂定义getter方法而不定义setter方法
class Student(object):

@property
def birth(self):
    return self._birth

@birth.setter
def birth(self, value):
    self._birth = value

@property
def age(self):
    return 2015 - self._birth
