HTTP协议

GET/HTTP/1.1
GET读取请求，/URL路径，HTTP/1.1 HTTP协议版本1.1,
HOST:www.xxx.com 域名
Context-Type: text/html浏览器判断相应内容

HTTP请求：

1 浏览器向服务器发送HTTP请求
方法：GET/POST
路径：/full/url/path;
域名：Host:www.xxx.com（POST请求包括Body）

2 服务器向浏览器返回HTTP响应：

响应代码：OK/3xx/4xx/5xx
响应类型：Content-Type
3可再次发出HTTP请求：
在HTML中可链入其他服务器资源，站点可链接其他站点

HTTP格式：Header和Body

GET/path HTTP/1.1
Header1: Value1
Header2: Value2
Header3: Value3

一行一个 \r\n

POST/path HTTP/1.1
Header1: Value1
Header2: Value2
Header3: Value3

body data

连续两个\r\n Header还Body

响应:

200 OK
Header1: Value1
Header2: Value2
Header3: Value3

body data

Content-Encoding：gzip Body数据压缩,解压缩获得数据

def application(environ, start_response):
    start_response('200 OK', [('Content-Type', 'text/html')])
    return [b'<h1>Hello, web!</h1>']

environ:HTTP请求信息的dict对象
start_response:发送HTTP响应函数

Web应用本质：

1浏览器发送一个HTTP请求；

2服务器收到请求，生成一个HTML文档；

3服务器把HTML文档作为HTTP响应的Body发送给浏览器；

4浏览器收到HTTP响应，从HTTP Body取出HTML文档并显示。

异步IO

当代码需要执行一个耗时的IO操作时，它只发出IO指令，并不等待IO结果，然后就去执行其他代码了。一段时间后，当IO返回结果时，再通知CPU进行处理。

同步IO，主线程挂起，异步IO，在消息循环中,主线程不断重复读取-处理消息，线程可处理多个IO请求。

loop = get_event_loop()
while True:
    event = loop.get_event()
    process_event(event)

asyncio

提供异步支持，异步操作在coroutine通过yield from完成，多个coroutine可封装成一组Task并发执行。 

import threading
import asyncio

@asyncio.coroutine
def hello():
    print('Hello world! (%s)' % threading.currentThread())
    yield from asyncio.sleep(1)
    print('Hello again! (%s)' % threading.currentThread())

loop = asyncio.get_event_loop()
tasks = [hello(), hello()]
loop.run_until_complete(asyncio.wait(tasks))
loop.close()

多个线程同时执行，当遇到asynico.sleep()线程不等待，中断进行下一个消息循环。返回时yield from拿到返回值（None）

@asyncio.coroutine把generator标记为coroutine类型.

with drain() 循环调度写入操作并刷新缓冲区。

aiohttp

基于asyncio实现的HTTP框架

import asyncio

from aiohttp import web

async def index(request):
    await asyncio.sleep(0.5)
    return web.Response(body=b'<h1>Index</h1>')

async def hello(request):
    await asyncio.sleep(0.5)
    text = '<h1>hello, %s!</h1>' % request.match_info['name']
    return web.Response(body=text.encode('utf-8'))

async def init(loop):
    app = web.Application(loop=loop)
    app.router.add_route('GET', '/', index)
    app.router.add_route('GET', '/hello/{name}', hello)
    srv = await loop.create_server(app.make_handler(), '127.0.0.1', 8000) #创建TCP服务
    print('Server started at http://127.0.0.1:8000...')
    return srv

loop = asyncio.get_event_loop()
loop.run_until_complete(init(loop))
loop.run_forever()
