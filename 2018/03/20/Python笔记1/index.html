Queue Pipes 实现进程间通信 

from multiprocessing import Process,Queue
    import os, time, random
def write(q):
    print('Process to write :%s' % os.getpid())
    for value in ['A', 'B', 'C']:
    print('Put %s to queue...' %value)
    q.put(value)
    time.sleep(random.random())

def read(q):
    print('Process to read: %s' % os.getpid())
    while True:
        value = q.get(True)
        print('Get %s from queue.' % value)

if __name__== '__main__':
    q = Queue()
    pw = Process(target=write, args=(q,))
    pr = Process(target=read, args=(q,))
        pw.start()
    pr.start()
    pw.join()
    pr.terminate()

6正则表达式
匹配

  • \w 字母、数字及下划线
  • \W 不是字母、数字及下划线的字符
  • \s 任意空白符，等价于[\t\n\r\f]
  • \S 任意非空字符
  • \d 任意数字[0-9]
  • \D 任意非数字的字符
  • \A 字符串开头
  • \Z 字符串结尾，存在换行，匹配换行前结束的字符串
  • \G 最后匹配完成的位置
  • \n 一个换行符
  • \t 一个制表符
  • ^ 一行字符串的开头
  • $ 一行字符串的结尾
  • . 任意字符除换行符，re.DOTALL标记被指定，匹配包括换行符任意字符
  • […] 表示一组字符
  • [^…] 不在[]中的字符
    • 0或多个表达式
    • 1或多个表达式
  • ？0或1个前面正则表达式定义的片段，非贪婪
  • {n} n个前面的表达式
  • {n,m} n到m次前面正则定义的片段，贪婪
  • a|b a或b
  • () 括号内的表达式，页表示一个组

修饰符

  • re.I大小写不敏感
  • re 本地化识别（locale-aware)匹配
  • re.M 多行匹配，影响^和$
  • re.S 使.匹配包括换行在内所有字符
  • re.U Unicode字符集解析字符。影响\w \W \b \B
  • re.X 给予更灵活的格式使正则容易理解

contextlib

enter exit实现上下文管理

from contextlib import contextmanager
class Query(object)


def __init__(self, name):
    self.name = name

def query(self):
    print('Query info about %s...' % self.name)

@contextmanager
def create_query(name):
    print('Begin')
    q = Query(name)
       yield q
        print('End')
with create_query('Bob') as q:
    q.query()

contextlib模块含有contextmanager对象实现上下文管理
@contextmanager装饰器decorator可接受生成器generator

@closing对象也可实现上下文管理 经过@contextmanager装饰的generator

from contextlib import closing
from urllib.request import urlopen

with closing(urlopen('https://www.python.org')) as page:
    for line in page:
        print(line)
@contextmanager
def closing(thing):
    try:
        yield thing
    finally:
        thing.close()

from xml.parsers.expat import ParserCreate
class DefaultSaxHandler(object):
def start_element(self, name, attrs):
    print('sax:start_element: %s, attrs: %s' % (name, str(attrs)))

def end_element(self, name):
    print('sax:end_element: %s' % name)

def char_data(self, text):
    print('sax:char_data: %s' % text)

handler = DefaultSaxHandler()
parser = ParserCreate()
parser.StartElementHandler = handler.start_element
parser.EndElementHandler = handler.end_element
parser.CharacterDataHandler = handler.char_data
