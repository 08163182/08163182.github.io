仓库创建

创建空目录
$mkdir learngit
$cd learngit
$pwd(显示)

初始化仓库
$git init

添加与提交文件
$git addr readme.txt
$git commit –m “…….”

查看仓库状态
$git status

查看修改痕迹
$git diff readme.txt

查看文件历史记录
$git log
$git log –pretty=oneline(精简信息)

文件版本回退

$git reset –hard HEAD^(^^)

$git reset –hard commit_id

记录每一次命令
$git reflog

查看版本内容
$cat readme.txt

版本库


$git add 工作区—-暂存区

$git commit 暂存区—-当前分支

撤销修改(回到最近一次commit或add时的状态)

$git checkout – readme.txt(撤销工作区修改)

$git reset HEAD readme.txt（撤销暂存区修改）

删除文件
$rm readme.txt

工作区和版本库不一致：

版本库删除并提交
$git rm readme.txt
$git commit –m”…..

远程仓库

创建ssh key
$ ssh-keygen –t rsa –C”your email@example.com”

本地仓库内容推送GitHub仓库
(先有本地仓库后有远程仓库)
$git remote add orign git@github.com:08163182/learngit.git

$git push –u origin master(第一次推送加-u关联远程库 默认名字origin)
$git push origin master

从远程库克隆
$git clone git@github.com:08163182/gitskills.git

分支管理

HEAD指向master master指向提交

创建并切换dev分支
$git checkout –b dev == $git branch dev
$git checkout dev

查看当前分支
$git branch

合并指定分支到当前分支
$git merge dev

删除分支
$git branch –d dev

合并分支的—no—ff模式

禁用Fast forward
$git merge –no—ff –m “…….” dev (-m因为创建了一个新的commit)

BUG分支

保护现场

$git stash

查看工作现场
$git stash list

恢复现场
$git stash pop

丢弃未合并分支
$git branch –D

查看远程库信息
$git remote
$git remote –v (详细信息)

多人协作

推送分支
$git push origin master
($git push origin dev)

推送失败$git pull(合并)
$git push origin master

$git branch –set-upstream master origin(本地分支与远程分支建立联系)

标签

创建标签

$git branch

$git checkout master

$git tag v1.0

命令标签
$
git tag v0.9 commit-id/HEAD
查看标签信息

$git show v0.9
$git tag –a v0.1 –m”….” 指定标签信息
$git tag –a v0.1 –m”….” PGP私匙签名标签
查看所有标签
$git tag
删除标签
$git tag –d v0,1
推送标签
$git push origin v1.0
–tags推送所有
$git push origin :refs/tags/删除远程标签

使用GitHub

1 GitHub Fork别人仓库
2 git clone git@github.com:08163182/bootstrap.git

git验证失败问题

设置ssh key：
配置信息：
git config --global user.name "你的GitHub用户名"
git config --global user.email "你的GitHub注册邮箱"
生成ssh密匙：
ssh-keygen -t rsa -C "你的GitHub注册邮箱"
ssh key公匙添加到Github上；

如果缺少known_hosts文件：
ssh-keyscan -t rsa github.com >> ~/.ssh/known_hosts

git push origin master --force强制推送
