语法

<?php
echo "Hello World!";
?>

echo输出文本;
变量对大小写敏感;
global声明全局变量；
static声明静态变量函数执行完不会被删除；
echo可输出多个字符串，print只能输出一个字符串，返回值为1；

php
=>类似指针，通常用于数组操作，array(key=>value)；
->用于类、对象操作，调用类方法；

EOF

echo <<<EOF
EOF;

EOF模式下单双引号自动转义；
var_dump()返回变量的数据类型和值

常量

define(name,value,case_insensitive)case_insensitive为true时忽略大小写，用define设置的变量为全局变量；

字符串

并置运算符.可连接字符串
strpos()在字符串中查找字符或文本并返回第一个匹配的字符位置；
x xor y异或；
x<=>y x与y比较分别返回一个小于等于大于0的值；
获取数组长度count();

数组

关联数组$age=array("Peter"=>"35")
遍历关联数组foreach($age as $x=$x_value){}

数组排序

sort() 升序 rsort()降序
asort()值升序 ksort()键升序
arsort()值降序 ksort()键降序

超级全局变量

$GLOABALS全局变量
$_SERVER获得服务器的一些项目
$_REQUEST收集HTML表单提交的数据
$_POST收集表单数据
$_GET收集表单数据或者收集URL发送数据

循环

遍历数组：
foreach($array as $value){}

魔术变量

  • LINE显示文件当前行号
  • FILE显示文件路径
  • DIR文件所在目录
  • FUNCTION函数名称
  • CLASS类的名称
  • TRAITTrait的名字
  • METGOD函数名
  • NAMESPACE命名空间

    命名空间

    namespace MyProject{}//命名空间内的代码
    namespace{}//全局代码
    命名空间可分层次例如namespace MyProject\Sub\Level;

    PHP表单

    GET/POST 收集表单数据；GET收集发送的信息对任何人可见（URL中），POST表单信息对其他人不可见（HTTP请求中 ）；

$_SERVER["PHP_SELF"]超全局标量可返回当前执行脚本的文件名；

htmlspecialchars()函数可把特殊字符转换为实体；<>转换为< 和>

trim()去除不必要字符；stripslashes()删除输入数据中的\

preg_match()函数检索字符串模式；

php正则表达

分隔符 // ## ~~
元字符:

  • \ 转义
  • ^ 行首
  • $ 行尾
  • . 匹配换行符外任何字符
  • [] 开始结束字符类定义
  • | 开始一个可选分支
  • () 子组的开始结束标记
  • ? 0或1次匹配
    • 0或多次匹配
    • 1或多次匹配
  • {} 自定义量词开始结束标记

\S匹配任意空白符 ^在[]表明字符类取反 - 标记字符范围；
\w匹配字母或数字或下划线

PHP数据库

php连接数据库
mysql_connect(servername,username,passwordd);
创建数据库
CREATE DATABASE
创建表
CREATE TABLE
mysql_query()用于向MYSQL连接发送查询或命令；
mysql_select_db()选择数据库；
$_GET收集表单值，对任何人可见
$_POST收集表单值，对任何人不可见
$_REQUEST收集GET和POST方法发送的表单数据

<?php
$con= mysql_connect("localhost","wjx","abc111");
if (!$con)
{
die('Could not connect:'.mysql_error());
}
mysql_select_db("my_db",$con);

$result = mysql_query("SELECT * FROM Persons
WHERE FirstName='Peter'");

while($row= mysql_fetch_array($result))
{
echo $row['FirstName'. " ". $row['LastName'];
echo "<br />";    
}    
mysql_close($con);
?>

php高级教程

日期

date(format,timestamp)

<?php
echo "今天是" . date("Y/m/d/h/i/sa"). "<br>";
?>

date_default_timezone_set()设置时区
mktime(hour,minute,second,month,day,year)创建日期
strtotime(time,now)字符串创建日期

包含

include ‘filename’; 错误发生脚本继续执行
require ‘filename’;错误发生停止执行

文件

fopen()打开文件
fclose()关闭文件
fgets()逐行读取文件
fgetc()逐字符读取文件
feof()文件末尾

Cookies

cookie远程浏览器识别用户
setcookie(name,value,expire,path,domain)
$_COOKIE取回cookie的值(发送cookie进行URL编码，取回解码)
setrowcookie()防止编码

Sessions

sessions()存储用户会话信息
session_start()开始
$_SESSION()存储和取回session变量
unset() `session_destroy()`销毁

E-Mail

mail(to,subject,messaage,headers,parameters)
（接收者，主题，发送消息，附加标题，额外参数）
FILTER_SANITIZE_EMAIL从字符串中删除电子邮件的非法字符
FILTER_VALIDATE_EMAIL验证电子邮件的值

错误处理

die()处理错误
error_function(error_level,err_message....)自定义错误处理器
tirgger_error()触发错误

异常处理

Try抛出异常
Throw触发异常
Catch捕获异常
每个try或throw至少对应一个catch
可在try代码块内的cathch代码块抛出异常

过滤器

Validating:验证用户输入
Sanitizing:允许或禁止字符串指定的字符
filter_var()过滤单一变量（一个过滤器)
filter_var_array()过滤多个变量（多个过滤器）
filter_input()获取输入变量过滤
filter_input_array()获取多个输入变量过滤

高级过滤器

<?php
$int= 122;
$min=1;
$max=200;
if(filter_var($int,FILTER_VALIDATE_INT,array("options=>array("min_range"=>$min, "max_range"=>$max)))===false)
{
echo("变量值不在合法范围内")
}
else
{
echo("变量值在合法范围内")
}
>?
