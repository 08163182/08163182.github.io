派生选择器

根据文档的上下文关系确定某个标签样式

DOM：标准文档对象模型
id属性为一个元素分配一个唯一的名字
同一HTML文档中，可使用多次class，只能使用一次id.
对代码进行模块化，可提高代码的复用性。
# sidebar p {}

id选择器

#red {color:red;}

类选择器

.center {text-align: center}
td.fancy {}

属性选择器

[title]
[title=W3School]指定值
[title~=hello]包含指定词汇
[lang|=en]以指定值开头
input[type="text"]
input[type="button"]

css创建

外部样式表：link链接到样式表，以.css存在

<head>
<link rel="stylesheet" type="text/css" href="mystyle.css" />
</head>

内部样式表：

<head>
<style type="text/css">
...
</style>
</head>

内联样式：

<p style="">
   ...
   </p>

多重样式:
内部样式与外部样式链接，颜色属性继承外部，文字排列和字体尺寸被内部样式表规则取代。

css样式

背景

背景颜色p {backgroud-color: gray;}
背景图像body {background-image: url(i/eg.gif);}
背景位置body {background-position:center;}
【center/top/bottom/right/left】
【50% 50%（图像对应点位）/50px 100px(左上角向右50，向下100偏移)】
背景关联background-attachment:fixed
背景不重复background-repeat:no-repeat;
背景属性不能继承

文本

p {text-indent: -5em; padding-left: 5em}实现文本缩进
text-align文本行水平对齐
justify水平对齐
p.spread{working-spacing: 30px;}字间隔
left-spacing字母间隔
text-transform(none/uppercase/lowercase/capitalize)字符转换
text-decoration(none/underline/overline/line-through/blink)值会替换而不累积
p {white-space: normal;}处理空白符（合并空白符忽略换行符）

  • pre 保留空白符合并换行符
  • nowrap 合并空白符忽略换行符
  • pre-wrap 保留空白符保留换行
  • pre-line 合并空白保留换行
    设置文本方向:
    unicode-bidi: embed/bidi-override
    direction:ltr/rtl

    字体

    字体类型：
    h1 {font-family: Georgia;}
    <p style="font-family: 'New York',Times;">...</p>
    字体风格：
    p.normal {font-style:normal;}#italic/oblique
    字体大小：
    h1 {font-size:60px;}
    em: pixels/16=em

    链接

  • a:link未访问的链接
  • a:visited用户已访问的链接
  • a：hover鼠标指针位于链接上方
  • a:active链接被点击时刻

列表

ul {list-style-type: square}
list-style-image
list-style-position
list-style

表格

table
    {
    border-collapse:collapse;
    }
table,th,td
    {
    border: 1px solid black;
    }
  • border-collapse 表格边框合并为单一边框
  • border-spacing 分割框距离
  • caption-side 标题位置
  • empty-cells 显示表格空单元格
  • table-layout 单元、行和列的算法

    轮廓

  • outline 轮廓属性
  • outline-color 轮廓颜色
  • outline-style 轮廓样式
  • outline-width 轮廓宽度

    框模型

    element: 元素
    padding: 内边距
    border: 边框
    margin: 外边距
    padding-top：上边距
    padding-right：右边距
    padding-bottom： 下边距
    padding-left： 左边距
    边框出现必须声明一个边框样式
    默认顺序 top-right-bottom-left
    p
    {
    border-style:solid;
    border-width:5px;
    }
    border-style/width/color/bottom
    border-bottom-color/style/width
    值赋值：
    上对下/左对右
    margin-top/right/bottom/left
    外边距合并：两个垂直外边距相遇将形成一个外边距

CSS定位

定位

display属性改变生成框的类型
CSS三种定位机制: 普通流、浮动、绝对定位
position:static/relative/absolute/fixed
overflow:元素内容溢出时的事件
clip: 元素形状
vertical-align: 元素垂直对齐方式
z-index:元素堆叠顺序

相对定位

\#box_relative{
    position: relative;
    left: 30px;
    top: 20px;
    }

相对定位“相对”元素在文档初始位置
绝对定位“相对”最近的已定位祖先元素

浮动

float元素浮动
clear包围元素视觉上包围浮动元素

选择器

选择器分组

通配符选择器：* {color:red;}
选择器可进行分组；
声明可进行分组；

类选择器

<h1 class="important">
This hedaing is vert important
</h1>
*.important {color:red;}
.important {color:red;}

ID选择器

#intro {font-weight: bold;}

ID只能使用一次，可包含更多含义

属性选择器

a[href][title] {color:red;}
planet[moons] {color:red;}
a[href="jasonwjx.com/"}[title:"Python"] {color:red;}

class~="def"包含指定词汇
abc^="def"以指定值开头
abc$="def" 以指定值结尾
abc*="def" 包含指定值
*[lang|="en"]{color:red;} 选择lang属性等于en或以en- 开头元素

后代选择器

h1 em {color:red;}

子元素选择器

h1 > strong {color:red;}
只在h1元素子元素的strong元素里有效

相邻选择兄弟选择器

h1 + p {margin-top:50px;}
选择紧接在h1元素后出现的段落
一个结合符只能选择两个相邻兄弟中的第二个元素
li + li {font-weight:bold;}

伪类

锚伪类：link-visited-hover-active固定顺序
伪类：selector: pseudo-class {property: value}
:active 被激活元素添加样式
:focus 拥有键盘输入焦点元素添加样式
:hover 鼠标悬浮在元素上方，向元素添加样式
:link 未被访问链接添加样式
:visited以访问
:first-child 元素的第一个子元素
:lang 带有指定lang属性

伪元素

:first-letter 首字母添加特殊样式
:first-line 首行添加特殊样式
:before元素前添加内容
:after 元素后添加内容

高级

对齐

margin 水平对齐 position 左右对齐 float 左右对齐

尺寸

height 元素高度 line-height行高 max-height 最大高度
max-width 最大宽度 min-width 最小宽度 width 宽度

分类

cursor 指向某元素之上显示的指针类型
visibility 元素是否可见

CSS媒介类型

@media 在相同样式表使用不同样式规则来针对不同媒介
screen： 电脑显示器
all: 所有媒介设备
handheld: 小的手持设备
projection: 方案展示
